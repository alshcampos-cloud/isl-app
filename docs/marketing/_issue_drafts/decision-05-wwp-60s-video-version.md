# Build "What's Worth Practicing" 60s video version (after audio-only v0.5)?

**Labels suggestion:** marketing, decision, creative-production, wave-2
**Assignees suggestion:** @lucas (owner) + @jacobdev-prog (test design)

---

## Context

WWP audio-only v0.5 is shipped (60s VO on waveform visualization, lives at `docs/marketing/mocks/wwp_audio_only_v0_5.mp4` per Launch Index). It's built for Reels/TikTok audio-discovery surfaces. A full video version — same script, with visuals — would open YouTube pre-roll + Reels/TikTok primary feed + LinkedIn video. Production cost estimate: $40-150 depending on AI-gen vs. shot footage.

## Options

- **A) Build 60s video using existing VO + recycled cinematic-pack β/ε footage** — Lowest-cost. β/ε have raw clips that didn't ship; repurpose under WWP voiceover. ~$0-40 production, mostly editor time.
- **B) Build fresh 60s video with new visuals (AI-gen via Veo/Sora)** — Higher production value, but burns §13 Verisimilitude Auditor token + risks aesthetic-default failures we've hit before. ~$80-150.
- **C) Don't build video — keep WWP as audio-only** — Audio has a clear job (audio-discovery). Video competes with Letter on Screen v2 + statics.
- **D) Build 30s + 15s audio cutdowns first** — Test whether shorter versions earn more shares. Cheaper, faster.

## Recommended

**A** — recycle the dead β/ε footage. Cheap, fast, gives us a 60s video without committing fresh production budget. If A earns traction, consider B. **D** is a fine parallel experiment (10-15 min editor time).
