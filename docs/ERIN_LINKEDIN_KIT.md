## 6. SCREEN RECORDING SCRIPT — Nursing Track Demo Video

### Purpose
A 45-55 second square video showing the nursing track in action. Post it alongside or shortly after the LinkedIn post. LinkedIn autoplays video muted, so text overlays carry the message.

### Pre-Recording Setup

1. **Browser:** Chrome, clean state
   - Hide bookmarks bar (Cmd+Shift+B on Mac)
   - Close all other tabs
   - No extensions visible

2. **Account:** Log into the beta account (the one with session history so dashboard stats aren't all zeros)

3. **Window size:** Resize browser to roughly 500px wide for a cleaner, mobile-like look
   - Or use Chrome DevTools → Toggle Device Toolbar → choose "iPhone 14 Pro"

4. **Recording tool:** QuickTime Player (Mac) → File → New Screen Recording → select the browser window area
   - Or use OBS (free) for more control

5. **Speed:** Navigate slowly and deliberately. Quick movements look jarring when compressed. Pause 1-2 seconds on each screen before moving.

Record these 10 shots in sequence. Don't stop between them — record it as one continuous take, then trim in editing.

### Post-Recording: Adding Text Overlays

**Recommended tool:** CapCut (free, available on Mac/iOS/web)
- Import your screen recording
- For each shot, add a text overlay:
  - **Font:** Clean sans-serif (e.g., Inter, Helvetica, or CapCut's default)
  - **Size:** Large enough to read on mobile
  - **Color:** White text
  - **Background:** Dark semi-transparent bar behind the text (CapCut calls this "text background" — set to black at 60-70% opacity)
  - **Position:** Bottom third of the frame
  - **Timing:** Match each overlay to its shot (use the shot list durations above)

**Export settings:**
- Aspect ratio: **1:1 (square)** — best for LinkedIn feed on desktop + mobile
- Resolution: **1080 x 1080**
- Format: MP4

**Upload:** Upload the video directly to your LinkedIn post (not as a YouTube/Vimeo link — native video gets 5x more reach on LinkedIn).
